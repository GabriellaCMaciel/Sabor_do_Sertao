# Sabor do Sertão

Site institucional para um delivery de comida nordestina, desenvolvido como projeto acadêmico para apresentação em sala do curso técnico em Desenvolvimento Web e Mobile.

---

## Objetivo

Criar uma página web que permita ao cliente conhecer o cardápio, montar um pedido e entrar em contato com o delivery de forma rápida e intuitiva.

## Público-alvo

- Consumidores de comida nordestina na região de Luziânia/GO
- Usuários que desejam pedir delivery pelo WhatsApp
- Pessoas que valorizam uma experiência visual acolhedora e simples
- Clientes que precisam de informações claras sobre localização, horário e contato

## Intenção do site

Este site tem a intenção de apresentar o negócio com profissionalismo e funcionalidade. Ele serve como uma vitrine digital para o serviço de delivery, facilitando a conversão de visitantes em pedidos reais.

## Problema que resolve

Muitos pequenos negócios de alimentação não possuem uma presença digital organizada. O site resolve esse problema ao:

- apresentar o cardápio de forma clara
- permitir montagem de pedido direto pela página
- disponibilizar contato e endereço de forma acessível
- reduzir o número de etapas necessárias para o cliente realizar um pedido

## Como resolve

A solução é implementada com uma página única e responsiva que concentra:

- navegação por seções (Início, Cardápio, Pedido, Contato)
- lista de itens e preços
- seleção de quantidade e adição ao carrinho
- envio automático de mensagem para WhatsApp com pedido formatado
- informações de telefone, local e horário de funcionamento

## Tecnologias utilizadas

- HTML5
- CSS3
- JavaScript

## Estrutura do projeto

- `index.html` — página principal e estrutura do conteúdo
- `src/styles/styles.css` — estilização do layout, tipografia e cores
- `src/js/main.js` — lógica do carrinho e envio de pedidos

## Como executar

1. Abra o arquivo `index.html` em um navegador moderno.
2. Navegue pelas seções usando o menu ou as âncoras da página.
3. Selecione um item no campo "Escolha o item" e ajuste a quantidade.
4. Clique em "Adicionar ao pedido".
5. Use o botão "Enviar Pedido pelo WhatsApp" para abrir a conversa com o pedido montado.
